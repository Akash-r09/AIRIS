# 🚀 Getting Started
Clone Repository
```
git clone https://github.com/Akash-r09/AIRIS.git
cd AIRIS
```
Backend
```
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```
Frontend
```
cd frontend

npm install

npm run dev
```
